Text Styling

Here we go again! in this module, we will learn how about change text inline styling with <span> and jQuery. The styling includes bold, italic, and both of them.

Example:

Hello World!

HTML:

<h2 id="sample-text"></h2>
<button id="btn-bold" onClick="make_bold()">Bold</h2>
<button id="btn-italic" onClick="make_italic()">Italic</h2>
<button id="btn-bold-italic" onClick="make_bold_italic()">Bold & Italic</h2>
<button id="btn-reset" onClick="reset()">Reset</h2>

jQuery:

function make_bold(){
   var sample_text = $("#sample-text");
   sample_text.attr("style", "font-weight:bold");
}
function make_italic(){
   var sample_text = $("#sample-text");
   sample_text.attr("style", "font-style:italic");
}
function make_bold_italic(){
   var sample_text = $("#sample-text");
   sample_text.attr("style", "font-weight:bold; font-style:italic");
}
function reset(){
   var sample_text = $("#sample-text");
   sample_text.removeAttr("style");
}
