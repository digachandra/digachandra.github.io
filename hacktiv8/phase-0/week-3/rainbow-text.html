Rainbow Text

Hi! we meet again..

Basically in this module, we will learn about how to change regular text color into rainbow (multiple color by the way) with jQuery. So lets start!

Example:

Hello World!

HTML:

<h2 id="rainbow-text"></h2>
<button id="btn-change-color" onClick="change_color()">Change Color</h2>

jQuery:

function change_color(){
   var pointer_text = $("#rainbow-text");
   var rainbow_color = ["#EE4035","#F37736","#FDF498","#7BC043","#0392CF"];
   var text_array = pointer_text.text().split("");
   var rainbow_text;

   for(var idx = 0; idx = text_array.length; idx++){
      var temp_text = "<span style=\"color:" + rainbow_color[Math.floor(Math.random()*rainbow_color.length)] + ";\">" + text_array[idx] + "</span>";
      if(idx == 0) rainbow_text = temp_text;
      else rainbow_text += temp_text;
   }
   pointer_text.html(rainbow_text);
}
