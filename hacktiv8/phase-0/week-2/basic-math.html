Basic Math

Hi! in this module, i will like to share how we can do a basic math with javascript and also one of well-known javascript framework in town jQUery 

var a = 2;
var b = 4;
var c = a + b;

Example Case:

a :
+
b :
=
c :

HTML:

a : <input type="number" id="case-1-a" onChange="case_1_calculate()" />
+ b : <input type="number" id="case-1-b" onChange="case_1_calculate()" />
= c : <input type="number" id="case-1-c" />

Javascript:

function case_1_calculate(){
   var value_a = parseInt(document.getElementById("#case-1-a").value));
   var value_b = parseInt(document.getElementById("#case-1-b").value));
   var pointer_c = document.getElementById("#case-1-c");
   if(value_a && value_b){
      pointer_c.value = value_a + value_b;
   } else (
      pointer_c.value = "";
   }
}

jQuery:

function case_1_calculate(){
   var value_a = parseInt($("#case-1-a").val());
   var value_b = parseInt($("#case-1-b").val());
   var pointer_c = $("#case-1-c");
   if(value_a && value_b){
      pointer_c.val(value_a + value_b);
   } else (
      pointer_c.val("");
   }
}

As we can see from the simple example above, the difference between javascript and jQuery only in the way we input and print out the value. Yap! basically jQuery is javascript framework that help us to minimize how we write javascript and maximize the capabilities of javascript itself with their own custom module.

We can learn what and how about all jQuery by following this link: api.jquery.com.
